# 🛡️ RAG 기반 정보보호 도메인 특화 LLM 시스템 설계

정보보호공학 지식 파트너 구축을 위한 핵심 도메인 가이드입니다.  
본 프로젝트는 **RAG(Retrieval-Augmented Generation)** 지식 구조를 통해 정보보호 개념을 체계적으로 정리하고, 연구자 수준의 고지능 답변을 제공하는 AI 파트너를 목표로 합니다.

## 🧩 도메인 집중 영역 (Coverage)

전문 지식은 크게 5가지 핵심 기둥(Pillar)으로 분류되어 관리됩니다.

- **Risk Management**: 자산 식별, 위협 분석(ISRM), 위험 제어

- **Cloud & Infrastructure**: CSPM, CWPP, 클라우드 공동 책임 모델
- **Ops & Engineering**: DevSecOps, 암호학, 보안 관제(SIEM/SOAR)
